05 — Accessibility

WCAG 2.2 AA Checks

PASS — Navy (#0A1F3C) on Gold (#D4A017)
Contrast ratio ≈ 8.2:1 — AA + AAA ✓
PASS — Navy (#0A1F3C) on White (#FFFFFF)
Contrast ratio ≈ 18.4:1 — AA + AAA ✓
PASS — White (#FFFFFF) on Blue (#1A5EBF)
Contrast ratio ≈ 5.3:1 — AA ✓
PASS — Gold (#D4A017) on Navy (#0A1F3C)
Contrast ratio ≈ 8.2:1 — AA + AAA ✓
FLAG — Muted (#6B7EA0) on White (#FFFFFF)
Contrast ≈ 3.8:1. Hanya untuk teks dekoratif / hint, bukan body copy.
Testable acceptance criteria
✓ Semua elemen interaktif dapat dicapai via Tab
✓ Focus ring gold visible minimal 3px (WCAG 2.2 3:1)
✓ Tidak ada outline:none tanpa pengganti
✓ Semua gambar punya alt text atau aria-hidden
✓ Error message terhubung via aria-describedby
✓ Toast menggunakan live region roles yang benar
✓ Warna bukan satu-satunya indikator state
✓ prefers-reduced-motion menonaktifkan transform
